# Router Docs

These docs cover the shipped router surface in `forge-signal-wasm`.

The router is a graph-native navigation product. It owns typed route
authoring, projection, admission, browser-history truth, recovery,
speculation, restore, replay, and the boundary artifacts that explain why a
route is visible now.
